MariFlow by SethBling
Find SethBling on YouTube at http://youtube.com/SethBling
Video about MariFlow: https://www.youtube.com/watch?v=Ipi40cb_RsI

This is a machine learning project which uses TensorFlow to train recurrent neural networks to play Super Mario Kart. The MariFlow Manual can be found at https://docs.google.com/document/d/1p4ZOtziLmhf0jPbZTTaFxSKdYqE91dYcTNqTVdd6es4/edit?usp=sharing
